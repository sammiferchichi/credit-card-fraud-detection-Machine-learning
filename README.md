# Credit Card Fraud Detection using Machine Learning

Credit Card Fraud Detection using Machine Learning in Python


This project focuses on detecting fraudulent credit card transactions using Machine Learning techniques in Python.

## Workflow
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Train/Test split
- Logistic Regression model
- Model evaluation

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Objective
To understand the end-to-end machine learning workflow and apply it to a real-world fraud detection problem.

## Model Performance
- Training Accuracy: 98.25%
- Test Accuracy: 93.18%
